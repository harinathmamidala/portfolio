HomeMate

A Full stack application that enables remote control and scheduling of devices using a web-based front-end. The front-end is built using React, a popular JavaScript library for building user interfaces. The back-end is powered by Node.js and Express, which provide a robust and scalable server-side architecture. The system connects to a real-time database that manages the state of each device and provides real-time updates to the front-end. Users can interact with the front-end to turn devices on and off, set schedules for specific devices, and monitor the status of their devices from anywhere with an internet connection. One of the key features of the project is the scheduling functionality, which allows users to set specific times for their devices to turn on and off. For example, a user could schedule their heater to turn on at 6:00 am and turn off at 8:00 am on weekdays. Overall, the project provides a user-friendly and convenient way for users to manage their IoT devices remotely. It demonstrates a strong understanding of modern web development technologies and IoT concepts, and has the potential to be expanded upon with additional features and integrations.
